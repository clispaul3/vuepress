# Linux
##　常用命令
   + ifconfig 查看网卡配置信息
   + ping www.baidu.com 检查网络是否连接
   + vi test.txt、a、esc、shift+zz
   + history 查看历史命令
   + cd - 切换到上一次目录
   + cd ~ 切换到用户的家目录
   + ls -lh
## window-cmd命令
   + 查看端口号占用情况：netstat -ano
   + 查看某个端口号是否被占用：netstat -aon|findstr "3000"
   + 关掉某个端口号占用的进程：taskkill /pid 1696 -t -f
