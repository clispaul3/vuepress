# 数组
   + 概念：数组能够在单个变量中存储多个值，数组元素的数据类型可以不一样
## 数组类型
   1. 值数组：带有数字 ID 键的数组
   ```
     $arr = [1,2,3,null,"php","name"=>"php"];
     $arr = array(1,2,3,4);
   ```
   2. 关联数组：带有指定的键的数组，每个键关联一个值
   ```
     $obj = ["name"=>"php","age"=>18];
     $obj = array(
         'name'=>'php'
     )
   ```
   3. 多维数组：包含一个或多个数组的数组
   ```
    $objArr = array(
        ['name'=>'php','age'=>18],
        ['name'=>'java','age'=>20],
        ['name'=>'web','age'=>10]
    );
   ```
