# 常用框架
## ThinkPHP
   1. 安装
   2. 配置虚拟主机
      + C:\Windows\System32\drivers\etc\hosts
      + httd.conf => Include conf/extra/httpd-vhosts.conf
      + httpd-vhosts.conf
   3. 配置分布式部署文件
      + .htaccess 文件 => 放在相应的目录
      ```
      Deny from all
      ```
      + httpd-vhosts.conf
      `Allowoverride all`
