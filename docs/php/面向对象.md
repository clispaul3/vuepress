# 面向对象
## 基本语法
   + 定义
```
    class Person {
        public $name;
        public $age;
        var $city;
        public function say(){
            echo 'my name is ' . $this->name;
        }
    }
    $ming = new Person();
    $ming->name = 'xiaoming';
    $ming->say();
    $ming->city = 'guangzhou';
    $class = 'Person';
    $niu = new $class();
    $niu->city = 'beijing';
    var_dump($ming);
    var_dump($niu);

    注意：1. 不带public的方法默认是public
         2. var 声明的变量是public
```
   + __construtor 构造方法
```
    class Person{
        public $name;
        var $age;
        function __construct($name,$age = 18){
            $this->name = $name;
            $this->age = $age;
        }
        function say($city){
            echo 'I am living in ' .$city;
        }
    }
    $ming = new Person('xiaoming',20);
    $niu = new Person('xiaoniu');
    $niu->say('guangzhou');
    var_dump($ming);
    var_dump($niu);
```
