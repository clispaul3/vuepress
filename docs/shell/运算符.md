# 运算符

## 算术运算符
  1. 表达式和运算符之间要有空格，例如 2+2 是不对的，必须写成 2 + 2
  2. 完整的表达式要被 ` ` 包含
  3. 在 MAC 中 shell 的 expr 语法是：$((表达式))，此处表达式中的 "*" 不需要转义符号 "\" 
```shell
 #!/bin/bash
 a=1
 b=2
 echo `expr $a + $b`
 # *是特殊字符，要先转义
 echo `expr $a \* $b`
 echo `expr $a - $b`
 echo `expr $a / $b`
 echo `expr $a % $b`
 echo `expr $a == $b`
 echo `expr $a != $b`
```
